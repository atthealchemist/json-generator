Usage
#attributeKey
pointer to object value by key attributeKey

Example:
{ "name": "Allen Heath", "title": "Mister #name" }
will translated into:
{ "name": "Allen Heath", "title": "Mister Allen Heath" }

@functionName
calling method functionName
List of supported methods
  • @index : number - generates index value
  • @firstName : string - generates random first name
  • @surName (alias @lastName) : string - generates random sur name
  • @array(fox, dog, cat) : array - generates array of sended arguments
  • @random : string - give random item of sended args

Example:
{ "firstName": "@firstName", "animal": "@random('dog','cat','turtle')" }
will translated into:
{ "firstName": "Annabel", "animal": "cat" }

^parentKey
get value of parent object by key

Example:
{ "id": "@index", "name": "Allen Heath", "title": "Mister #name" "contacts": { "forId": "^id" } }
will translated into:
{ "id": 256, "name": "Allen Heath", "title": "Mister Allen Heath", "contacts": { "forId": 256 } }

__HIDDENOBJECT
hides object from output, but can access all data of this object

Example:
{ "id": "@index", "name": "Allen Heath", "title": "Mister #name", "contacts": { "forId": "^id" } "__HIDDEN": { "child": "Satoko" }, "possibleChildName": "#__HIDDEN.child" }
will translated into:
{ "id": 256, "name": "Allen Heath", "title": "Mister Allen Heath", "contacts": { "forId": 256 }, "possibleChildName": "Satoko" }

@repeat(arrayName, start, end)
Generates array with random count between start and end

Example:
{ "id": "@index", "name": "Allen Heath", "title": "Mister #name", "contacts": { "forId": "^id" } "__HIDDEN": { "child": "Satoko" }, "@repeat(children, 1, 3)": [ { "id": "@index", "name": "@firstName", "father": "^name" } ] }
will translated into:
{ "id": 256, "name": "Allen Heath", "title": "Mister Allen Heath", "contacts": { "forId": 256 }, "possibleChildName": "Satoko", "children": [ { "id": 0, "name": "Satoko", "father": "Allen Heath" }, { "id": 1, "name": "Sid", "father": "Allen Heath" } ] }
